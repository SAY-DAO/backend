### Requirements

- [Docker](https://docs.docker.com/get-docker/)
- [Docker-Compose](https://docs.docker.com/compose/install/)

### Run

#### Server

```bash
./scripts/up.sh
```
    Server Address: http://127.0.0.1:5000/

#### Tests
```bash
./scripts/test.sh
```

#### Shell
```bash
./scripts/bash.sh
```
